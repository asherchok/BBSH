# BigBack Steakhouse - Bug Reports & Feedback

When posting an issue, please include:

1. **What happened** - What went wrong or felt off?
2. **What you expected** - What should have happened instead?
3. **Steps to reproduce** - How can I trigger it again?
4. **Screenshot or video** - If possible, attach one.
5. **Version number** - Check bottom left of main menu.

For general feedback or suggestions, just describe what's on your mind. All feedback is welcome!
